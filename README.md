# 寄稿ルール
### ドメイン: DreamHackers/hackathon
#### ハッカソン自体の内容を記す: practice
`{YYYY}/hackathon{連番}_{MMdd}`  
cf.) practice/2015/hackathon00_0401md

#### 資料が必要な時にディレクトリを作成して置く
`{YYYY}/hackathon01/hoge.txt`

# ディレクトリ階層
* `practice`: ハッカソン活動内容
* `document`: ドキュメントメモなど
